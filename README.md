# funcoes-excel
Relatório interativo em Excel com análise de filmes 2007–2011 — funções avançadas, dashboard e storytelling de dados.
# 🎬 Análise de Filmes 2007–2011 | Excel Avançado

Projeto desenvolvido como entrega da **Semana 7** do programa de 
Capacitação em Ciência de Dados da [Asimov Jr](https://asimovjr.com.br/), 
com foco em proficiência no Excel — desde o tratamento da base até 
a construção de um dashboard interativo.

## 📊 Sobre o projeto

A base de dados contém informações de filmes lançados entre 2007 e 2011,
incluindo gênero, Audience Score, Rotten Tomatoes % e bilheteria mundial.
O objetivo foi transformar esses dados brutos em um relatório estruturado
e visualmente informativo, aplicando as principais ferramentas do Excel.

## 🗂️ Estrutura do arquivo

| Aba | Descrição |
|---|---|
| `Movie Data` | Base de dados original com 74 filmes |
| `Resumo` | Tratamento da base: duplicatas, texto para colunas, filtros |
| `Analysis` | 8 blocos de fórmulas cobrindo todas as categorias do curso |
| `Dashboard` | Relatório visual com KPIs, rankings e tabelas comparativas |

## ⚙️ Funções e recursos utilizados

**Estatísticas:** `SUM` `AVERAGE` `MAX` `MIN` `LARGE` `SMALL` 
`COUNTA` `COUNTBLANK`

**Lógica:** `IF` `OR` `IFERROR`

**Condicionais:** `COUNTIF` `SUMIF` `AVERAGEIF` 
`COUNTIFS` `SUMIFS` `AVERAGEIFS`

**Pesquisa:** `VLOOKUP` `MATCH` `INDEX`

**Texto:** `TRIM` `UPPER` `CONCATENATE` `LEFT` `RIGHT` `MID`

**Data:** `TODAY` `YEAR` `MONTH` `DAY`

**Outros recursos:** formatação condicional, validação de dados, 
congelamento de painéis, remoção de duplicatas, barras de dados

## 💡 Principais insights

- Comédias dominam em volume (41 filmes), mas Animações lideram
  em bilheteria média
- A diferença entre Audience Score e Rotten Tomatoes revela gêneros
  que dividem público e crítica
- O Top 10 de bilheteria é concentrado em poucos gêneros,
  sugerindo padrões de consumo do período
